Итоговый тест

1.Асимптотический анализ - это…
a.Space Complexity
b.World Complexity
c.Time Complexity
d.Galaxy Complexity
 Ответ:c.Time Complexity,a.Space Complexity


2.Какой алгоритм быстрее?
a.O(1000 n log n)
b.O(10 n log n)
c.O(2 log (1000n))
d.O(n)
 Ответ:c.O(2 log (1000n))

3.Какой базовый случай для рекурсии будет работать, если 
n >= 30 && n < 100?
a.int f(int n) if (n <= 1) return 1; else return n*f(n - 1)
b.int f(int n) if (n == 100) return 1; else return n*f(n - 1)
c.int f(int n) if (n == -1) return 1; else return n*f(n + 1)
 Ответ:a.int f(int n) if (n <= 1) return 1

4.Выберите анализ алгоритма, который оценивает алгоритм в лучшем случае.
a.Big O notation
b.Big Omega notation
c.Big Theta notation
 Ответ:b.Big Omega notation

5.Верно ли утверждение: существует только один возможный алгоритм для конкретной задачи.
a.Верно
b.Неверно
Ответ:b.Неверно

6.Если вы используете линейный поиск в массиве «Ar», какое значение будет наихудшим?
	Ar = | 3 | 4| 1| 9| 6| 2| 7|
a.2
b.3
c.7
d.9
Ответ:c.7

7.Выберите алгоритм O(n^2)?
a.Сортировка слиянием
b.Сортировка пузырьком
c.Линейный поиск
d.Быстрая сортировка
e.Бинарный поиск
Ответ: b.Сортировка пузырьком

8.Какая из следующих структур данных НЕ является линейной структурой данных?
a.Массивы
b.Связанные списки
c.Деревья
d.Стек
Ответ:c.Деревья

9.Функция, которая находит местоположение элемента с заданным значением - это:
a.Обход
b.Поиск
c.Сортировка
Ответ:b.Поиск

10.Верно ли утверждение: в красно-черном дереве у черного узла должны быть только красные потомки.
a.Верно
b.Неверно
Ответ:b.Неверно

11.Что из следующего верно для красно-черного дерева?
a.Корень красный
b.Корень и листья черные.
c.Все деревья зеленые, правильного ответа нет.
d.Каждый узел либо красный, либо черный.
e.Каждый лист (NULL) черный.
Ответ:b,d,e

12.Стек - это…
a.First in, first out
b.First in, last out
c.Last in, last out
d.Last in, first out
Ответ: d.Last in, first out

13.Сколько итераций потребуется для сортировки следующего массива с использованием алгоритма пузырьковой сортировки?
Ar = |4|5|7|6| 
a.2
b.3
c.4
d.5
c.6
Ответ:b.3

14.Очередь (queue) НЕ поддерживает следующую операцию:
a.Вставка
b.Удаление
c.Восстановление
d.Обход
Ответ:c.Восстановление

15.Допустимые значения высоты дерева AVL:
a.-2, 0, 2
b.0
c.0, 1
d.-1, 0, 1
Ответ: a.(-2, 0, 2)

16.Что из перечисленного является двойным вращением?
a.LL
b.LR
c.RR
d.LRL
Ответ: b.LR

17.Последовательность вставки 1, 2, 4 приведет к дисбалансу. Какое вращение можно использовать для балансировки дерева?
a.RR
b.LL
c.LR
d.RL
Ответ: c.LR

18.Какова временная сложность подсчета количества элементов в связанном списке?
a.O(n)
b.O(n log n)
c.O(1)
d.O(log n)
Ответ: a.O(n),c.O(1)

19.Класс Связанного списка реализуется из…
a.Interface Array
b.Interface Object
c.Interface Map
d.Interface List
Ответ: d.Interface List

20.Учитывая следующий метод, что вернет тест, если 
a = {0, 2, 3, 4} and v = 1?

a. 1
b.-1
c. 0
d. 4
Ответ: b.-1

21.Верно ли утверждение: алгоритм бинарного поиска можно использовать только в том случае, если данные упорядочены.
a.Верно
b.Неверно
Ответ: a.Верно 

22.Верно ли утверждение: пузырьковая сортировка — это эффективный алгоритм, реализующий подход «разделяй и властвуй».
a.Верно
b.Неверно
Ответ: b.Неверно

23.Линейный поиск — это эффективный алгоритм, реализующий подход «разделяй и властвуй».
a.Верно
b.Неверно
